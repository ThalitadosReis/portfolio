### Installation

**Clone the repository:**
   ```bash
   git clone https://github.com/ThalitadosReis/portfolio.git
   cd portfolio
   ```

**Install the dependencies:**
   ```bash
   npm install
   ```

**Start the development server:**
   ```bash
   npm run dev
   ```
